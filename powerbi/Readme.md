# Power BI Dashboard

## Overview

This dashboard provides an executive overview of European flight traffic, 
using data processed through an AWS serverless analytics pipeline.

## Dashboard Features

- Executive KPI Cards
- Top 10 Airports by Flight Traffic
- Monthly Flight Traffic Trend
- Top 10 Countries by Flight Traffic
- Top 10 Countries by IFR Flight Movements
- Interactive Slicers
  - Year
  - Month
  - Country
  - Airport
